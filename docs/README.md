# Background Documentation

This directory contains explanatory materials, non-normative rationale, and historical context for AEBS.

## Contents

- `rationale.md` — Design rationale and motivation
- `scope.md` — Scope and boundaries of AEBS
- `minimal-spec.md` — Minimal specification overview

## Purpose

These documents provide:
- **Context** — Why AEBS exists
- **Rationale** — Design decisions and trade-offs
- **Scope** — What AEBS does and doesn't address

## Normative vs Non-Normative

**Normative** (authoritative): `/spec/`
**Non-normative** (explanatory): `/docs/` (this directory)

For implementation, refer to `/spec/`.
For understanding, refer to `/docs/`.
